🌱 #Sign‑Up Form Project
A simple practice sign‑up form built with HTML, CSS, and basic validation attributes.
The design includes a background image, a logo styled with the Norse font, and a two‑column layout for user input.

------------------------------------------------------------------------

✨ #Features
- Two‑column layout using Flexbox for clean structure.
- Custom Norse font imported via CDN for branding.
- Form validation:
- Email field requires proper format.
- Phone field uses type="number".
- Password fields require at least 5 characters and must match a letter‑only pattern.
- Visual feedback with :valid and :invalid selectors.
- Styled button with hover/active states.
- Background and logo images integrated for design.

------------------------------------------------------------------------

🚀 #Getting Started
- Clone or download the repository.
- Place grass.jpg and Logo.png in the same directory as index.html.
- Open index.html in your browser.
- Fill out the form to test validation rules.

------------------------------------------------------------------------

🖱️ #Controls
- First Name / Last Name → Enter your personal details.
- Email → Must be in valid email format.
- Phone Number → Accepts numeric input.
- Password / Confirm Password → At least 5 letters, must follow the pattern.
- Create Account Button → Submits the form (currently no backend).

------------------------------------------------------------------------

🛠️ #Technologies Used
- HTML5 for structure
- CSS for styling and layout
- Validation attributes (minlength, pattern, type) for basic checks
